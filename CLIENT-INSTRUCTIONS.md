## 🚀 How to Use

1. **Bookmark** one or both links above (press Ctrl+D / Cmd+D)
2. **Open the link** in your browser (Chrome, Firefox, Safari, or Edge)
3. **Click "Choose File"** and select your DRF file from your computer
4. View interactive charts or generate PDF reports

---

## 🔄 Getting Updates

**You don't need to download anything!**

When updates are available:
1. Simply **refresh your browser** (Press F5 or Cmd+R / Ctrl+R)
2. You'll automatically get the latest version

That's it! No downloading, no file management, no installation.

## 🆘 Troubleshooting

**If the page doesn't load:**
- Check your internet connection
- Try clearing your browser cache (Ctrl+Shift+Delete / Cmd+Shift+Delete)
- Try a different browser

**If DRF file won't upload:**
- Make sure it's a valid DRF file format
- Try refreshing the page and uploading again
